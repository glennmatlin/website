# Suggested Commands

## Development
- `quarto preview` - Start dev server with live reload (localhost:4000)
- `quarto render` - Build the full site to _site/

## Deployment
- `quarto publish gh-pages` - Deploy to GitHub Pages

## Git
- `git status` - Check working tree
- `git add <files>` - Stage files
- `git commit -m "message"` - Commit
- `git push` - Push to remote

## Python (if needed)
- `uv sync` - Install dependencies
- `uv run ruff check .` - Lint Python code
- `uv run ruff format .` - Format Python code
- `uv run pyright` - Type check Python code

## System (macOS/Darwin)
- `ls`, `find`, `grep` - Standard unix utilities
- `open .` - Open current directory in Finder
- `pbcopy` / `pbpaste` - Clipboard operations
