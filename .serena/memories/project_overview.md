# Project Overview

## Purpose
Academic website for Glenn Matlin, a PhD CS researcher at Georgia Tech. The site showcases research (RAPID-AI dissertation), publications, projects, blog posts, and a CV.

## Tech Stack
- **Static site generator:** Quarto (https://quarto.org)
- **Content format:** Quarto Markdown (.qmd files)
- **Configuration:** YAML (_quarto.yml)
- **Styling:** CSS (styles/custom.css) with Bootstrap themes (cosmo/darkly)
- **Deployment:** GitHub Pages via `quarto publish gh-pages`
- **Python tooling:** uv (package manager), ruff (linter), pyright (type checker)

## Site URL
- Production: https://glennmatlin.com
- GitHub: https://github.com/gmatlin/website

## Key Design
- Blue Angels-inspired theme (navy #003087, gold #D4AF37)
- Dark mode support
- Responsive design
- Google Fonts (Inter)
