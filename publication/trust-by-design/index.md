## Abstract

BELLA (Budget-Efficient LLM Selection via Automated skill-profiling) is a framework that recommends optimal LLM selection for tasks through interpretable skill-based model selection, enabling practitioners to make principled and cost-performance trade-offs for deploying LLMs.
