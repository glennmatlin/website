## Abstract

FinForge introduces a framework for semi-synthetic financial benchmark generation, enabling the creation of rigorous and scalable evaluation benchmarks for language models in the financial domain. The approach combines authoritative financial data sources with synthetic generation techniques to produce high-quality evaluation datasets.
